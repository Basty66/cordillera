# Arquetipos Maven

## ms-service-archetype
Arquetipo para microservicios Spring Boot con JPA + PostgreSQL

## bff-archetype
Arquetipo para modulos BFF con Spring Boot + Security + JWT + Resilience4j

## Uso

mvn archetype:generate -DarchetypeGroupId=com.grupocordillera -DarchetypeArtifactId=ms-service-archetype -DarchetypeVersion=1.0.0
